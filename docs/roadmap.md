# Roadmap

## Phase 1 — Foundation
- FastAPI backend and core orchestration APIs
- Email intent extraction and draft response generation
- Calendar event creation and team coordination basics

## Phase 2 — Enterprise Integration Depth
- Microsoft 365 / Outlook / Teams production adapters
- Business Central, SharePoint, and CRM connectors
- Approval workflow and governance controls

## Phase 3 — Autonomous Operations
- Follow-up SLA automation and prioritization intelligence
- Voice escalation via Twilio/Vapi
- MCP tooling for enterprise knowledge routing

## Phase 4 — AI-Native Operations Fabric
- Long-term memory and context graph
- Multi-agent planning and delegation
- Advanced analytics and executive command dashboard
